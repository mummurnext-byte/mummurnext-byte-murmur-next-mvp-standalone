# AGENTS.md instructions

## Communication / Progress Updates

Progress updates should avoid optional user-facing commentary. Keep them concise and task-relevant. Include only necessary assumptions, decisions, blockers, verification results, and final outcomes.

## Sprint Product Demo

After each Sprint is completed, automatically prepare a Product Demo that includes:

- Deploy the completed Sprint to a Vercel Preview.
- Generate test data for the demo environment.
- Create one Demo Digital Human.
- Create one Demo Song.
- Create one Workflow.
- Create Dashboard data for review.
